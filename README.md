\# CILabProject



A simple Maven-based Java project demonstrating Continuous Integration using Jenkins.



\## Tools Used

\- Java 17

\- Maven 3.9.12

\- Jenkins

\- GitHub

\- JUnit



\## CI Pipeline Stages

1\. Checkout source code from GitHub

2\. Build using Maven

3\. Run unit tests automatically



\## Result

Build and tests run successfully via Jenkins pipeline.
